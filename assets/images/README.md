# 📸 תיקיית תמונות — ARC Residence

לכאן מעלים את 5 התמונות של הפרויקט. **חשוב מאוד:** תן לכל קובץ בדיוק את השם שמופיע בטבלה, ככה הוא ייכנס אוטומטית למקום הנכון באתר.

| שם הקובץ (חובה מדויק) | התמונה שלך | איפה מופיע באתר |
|---|---|---|
| `hero.jpg` | מגדל ARC בשקיעה על רקע ההרים | רקע ראשי גדול בראש הדף |
| `interior.jpg` | סלון/דירת היוקרה מבפנים | כרטיס "דירות יוקרה" בגלריה |
| `balcony.jpg` | המרפסת עם נוף היער | כרטיס "מרפסות עם נוף" בגלריה |
| `plaza.jpg` | כיכר הפרויקט עם המיתוג ARC | כרטיס "כיכר הפרויקט" בגלריה |
| `lifestyle.jpg` | הגינה עם המשפחות / האדם עם הכלב | קטע "סביבת מגורים" |
| `logo.png` | (אופציונלי) לוגו ARC ברקע שקוף | תפריט עליון |

## איך מעלים דרך GitHub (ידני)
1. היכנס לריפו ב-GitHub → ודא שאתה על הברנץ' **`claude/storage-project-inquiry-zed9ie`**.
2. נווט לתיקייה `assets/images/`.
3. לחץ **Add file → Upload files**.
4. גרור את 5 התמונות (אחרי ששינית להן את השמות לפי הטבלה).
5. **Commit** ישירות לברנץ'.

ברגע שתעלה — תגיד לי ואני אמשוך, אוודא שהכול מתחבר נכון, ואשלח לך צילום מסך של האתר. ✨

# 📸 תיקיית תמונות לאתר

זאת התיקייה שאליה מעלים את כל התמונות של האתר.

## איך מעלים?
פשוט גוררים לכאן את קבצי התמונות (או מצרפים לי אותן בצ'אט ואני אשים אותן כאן).

## שמות קבצים מומלצים (כדי שאשלב אותן באתר בקלות)
- `hero.jpg` — תמונת הרקע הגדולה בראש הדף
- `property-1.jpg`, `property-2.jpg`, `property-3.jpg` — תמונות נכסים לגלריה
- `about.jpg` — תמונה לקטע "עלינו"
- `logo.png` — לוגו (רקע שקוף עדיף)

## פורמטים נתמכים
`.jpg` / `.jpeg` / `.png` / `.webp` / `.svg`

> טיפ: תמונות באיכות טובה אבל לא ענקיות (עד ~500KB לתמונה) ייתנו אתר מהיר.
